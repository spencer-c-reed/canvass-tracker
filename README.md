# Canvass — Production-Standard Tracker

Live, auto-updating progress board for the election-law corpus production
campaign. Sanitized metadata only: counts, queue depth, runner state,
gate/criteria status, and commit subjects. No legal content and no
credentials are ever written here.

- Dashboard: index.html (GitHub Pages)
- Live data: status.json (regenerated every 10 min by tools/push_status.sh)
- Assessment (slow-changing): assessment.json

The source corpus and pipelines live in a separate private repo.
